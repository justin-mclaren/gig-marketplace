# gig-marketplace

## Tools Installed

### Specify CLI
GitHub Spec Kit - Spec-Driven Development Toolkit has been installed via:
```bash
uv tool install specify-cli --from git+https://github.com/github/spec-kit.git
```

The `specify` command is now available with the following features:
- `specify init` - Initialize a new Specify project from the latest template
- `specify check` - Check that all required tools are installed
